# Reflow Algorithm Design Prompts

## Dependency Handling
Prompt used to reason about multi-parent dependencies and scheduling order.

"Design a scheduling algorithm that respects work order dependencies,
work center conflicts, and shift boundaries."

## Shift Calculation
Used AI assistance to validate edge cases when work spans shift boundaries
and resumes next working day.

## Maintenance Handling
Clarified approach for handling fixed maintenance work orders and
maintenance windows without rescheduling.
